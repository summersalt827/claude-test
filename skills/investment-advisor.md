# 投资H技能模板（investment-advisor）

## 技能目标
为 Helen 生成**当日 A 股投资建议**，并将结果沉淀到 `investment/YYYY-MM-DD.md`，确保建议有依据、可执行、含风险提示。

## 输入来源
- `investing.md`：投资偏好、风险承受能力、关注板块/标的
- `profile.md`：个人背景与决策偏好
- `daily_news/YYYY-MM-DD.md`：新闻H输出（可选参考）
- 当日市场信息：政策、指数、资金面、板块与个股动态

## 输出规范
- 输出文件：`investment/YYYY-MM-DD.md`
- 格式：Markdown
- 长度建议：600-1200字
- 必含模块：
  - 市场判断
  - 重点板块
  - 具体建议（买入/持有/减仓/观望）
  - 风险提示
  - 投资逻辑
  - 免责声明

## 工作流程（执行顺序）
1. 读取配置：先读取 `investing.md`，明确风格与约束。
2. 收集信息：整理当日政策、指数、资金与公告信息。
3. 建立结论：先给大盘结论，再给板块优先级。
4. 形成建议：按“可执行动作 + 条件 + 风险边界”输出。
5. 落盘归档：写入 `investment/YYYY-MM-DD.md`。

## 建议模板（可直接复用）
```markdown
# A股投资建议 - YYYY-MM-DD

## 📊 市场整体判断
- [一句话结论]
- [核心依据1]
- [核心依据2]

## 🎯 重点关注板块
1. [板块A]： [原因]
2. [板块B]： [原因]

## 💹 具体操作建议
- [标的/ETF1]： [买入/持有/减仓/观望]；触发条件：[价格/量能/事件]
- [标的/ETF2]： [动作]；触发条件：[...]

## ⚠️ 风险提示
- [系统性风险]
- [行业/个股风险]
- [仓位与止损提醒]

## 💡 投资逻辑
[3-5条简要逻辑，说明“为什么这么做”]

---
数据来源：[新闻/公告/公开市场数据]
分析时间：[YYYY-MM-DD HH:mm]
*免责声明：以上内容仅供参考，不构成投资建议，投资有风险，入市需谨慎。*
```

## 质量检查清单
- [ ] 结论是否与 Helen 偏好一致
- [ ] 是否给出可执行动作而非空泛判断
- [ ] 是否明确风险与仓位控制
- [ ] 是否包含免责声明
- [ ] 是否已保存到正确日期文件
