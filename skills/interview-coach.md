# 面试H技能模板（interview-coach）

## 技能目标
帮助 Helen 跟踪求职进展、准备面试、复盘反馈，并把当日记录保存到 `interview/YYYY-MM-DD.md`。

## 输入来源
- `profile.md`：背景、优势、职业偏好
- 用户当天反馈：是否有面试、是否有投递、目标岗位
- 历史记录：`interview/YYYY-MM-DD.md`（如存在）

## 输出规范
- 输出文件：`interview/YYYY-MM-DD.md`
- 格式：Markdown
- 必含模块：
  - 今日状态（面试/投递/无安排）
  - 面试准备建议
  - 常见问题与回答要点
  - 下一步行动清单

## 工作流程（执行顺序）
1. 先确认状态：今天有面试、仅投递，还是暂无安排。
2. 收集关键信息：公司、岗位、时间、面试轮次、形式。
3. 输出针对性建议：按岗位能力模型给准备建议。
4. 给行动清单：按“今天/明天/本周”拆分。
5. 落盘归档：写入 `interview/YYYY-MM-DD.md`。

## 三种场景模板

### 场景A：今天有面试
```markdown
## 今日面试信息
- 公司/岗位：[...]
- 时间/形式：[...]
- 轮次：[...]

## 面试前准备
- 自我介绍（1分钟 + 3分钟版本）
- 项目案例（STAR结构）
- 岗位高频问题清单
- 反问面试官问题（2-3个）

## 临场建议
- 开场30秒策略
- 回答结构（结论先行 + 证据）
- 不会问题的应对话术
```

### 场景B：今天有投递
```markdown
## 今日投递进展
- 已投递岗位：[...]
- 匹配度评估：[高/中/低]

## 简历优化建议
- 标题与关键词优化
- 关键成果量化
- 项目经历贴合JD

## 后续跟进行动
- [ ] 24小时内补充作品集/案例
- [ ] 3天后礼貌跟进
```

### 场景C：暂无安排
```markdown
## 当前状态
- 暂无面试或新投递

## 本周求职策略
- 目标岗位列表更新
- 每日投递目标
- 模拟面试安排
- 技能补强任务
```

## 每日记录模板（可直接复用）
```markdown
# 求职面试记录 - YYYY-MM-DD

## 📌 今日状态
[面试 / 投递 / 暂无安排]

## 🎯 重点建议
- [...]
- [...]

## 💬 高频问题准备
1. [问题1]： [回答要点]
2. [问题2]： [回答要点]

## ✅ 下一步行动
- 今天：[...]
- 明天：[...]
- 本周：[...]
```

## 质量检查清单
- [ ] 建议是否针对具体岗位而不是泛化建议
- [ ] 是否包含可执行动作和时间节点
- [ ] 是否体现 Helen 个人背景优势
- [ ] 是否完成当日记录落盘
