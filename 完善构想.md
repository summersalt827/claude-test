# Helen的AI生活助理团队 - 完善构想

## 项目概述
打造5个AI Agent作为Helen的数字分身，构建个人助理团队

## Agent角色设计

### 1. 新闻H (NewsH)
**功能：**
- 每天早上收到"早上好"后自动获取AI相关新闻
- 总结成新闻简报
- 保存到 `daily_news/YYYY-MM-DD.md`
- 


**触发条件：** 用户说"早上好"/"早"/morning等
**输出格式：** Markdown新闻简报

### 2. 投资H (InvestH)  
**功能：**
- 在新闻H之后询问是否查看A股盘前早报
- 获取东方财富盘前新闻
- 结合investing.md文件给出投资建议
- 保存到 `investment/YYYY-MM-DD.md`

**触发条件：** 新闻H完成后自动询问
**依赖文件：** `investing.md` (投资策略/偏好)

### 3. 面试H (InterviewH)
**功能：**
- 询问今天是否有面试或投递简历
- 结合用户回答提供面试准备建议
- 记录面试日程和准备情况

**触发条件：** 投资H完成后自动询问
**输出：** `interview/YYYY-MM-DD.md`

### 4. 生活H (LifeH) - 待补充
**功能：** 日程管理、待办事项、生活提醒

### 5. 学习H (StudyH) - 待补充  
**功能：** 学习计划、知识整理、技能提升

## 需要的MCP服务

### 必需MCP
1. **Web搜索MCP** - 获取新闻和市场数据
2. **文件系统MCP** - 读写本地文件
3. **日历MCP** - 管理面试日程
4. **飞书MCP** - 发送消息到飞书

### 可选MCP  
5. **股票API MCP** - 获取实时A股数据
6. **邮件MCP** - 处理简历投递相关邮件
7. **提醒MCP** - 设置定时提醒

## 工作流程

```
早上好 → 新闻H → 投资H → 面试H → (可选)生活H/学习H
```
